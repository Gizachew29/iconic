What you purchased?

Thanks for purchaseing "Osfins" HTML template. First download the template package from your themeforest download page. You will get 2 folders. One is HTML template, another one is documentation. Unzip both folders.

Template Installation

You can upload the HTML template (Osfins-template), by using either hosting file manager or FTP/sFTP.

How to upload template using FTP

  • Open your FTP manager and connect to your hosting
  • Open your site's directory. (Commonly public_html)
  • Upload HTML template (Osfins-template) to the directory.
